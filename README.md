# CargoPro Backend System

## Setup Instructions
1. Clone the repo
2. Set up PostgreSQL and configure application.properties
3. Run the application using Maven or your IDE
4. Access Swagger UI at `/swagger-ui.html`

## API Overview
- Load Management
- Booking Management

## Technologies
- Spring Boot 3
- PostgreSQL
- Spring Data JPA
- Swagger / OpenAPI
- JUnit / Mockito
